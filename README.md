## Projects

### PDF Toolkit
A command-line utility built using `pypdf`.

**Features**
- Merge multiple PDFs
- Split PDFs into two parts
- Count PDF pages
- Extract text from PDFs
- View PDF metadata

**Concepts Used**
- Functions
- File Handling
- Exception Handling
- External Libraries (`pypdf`)

## Requirements

Python 3.10+

Install dependencies:

```bash
pip install -r requirements.txt
```

Or install project-specific libraries manually.

Example:

```bash
pip install pypdf
```

---

## Running a Project

Navigate to the project folder and run:

```bash
python main.py
```

Example:

```bash
cd PDF-Toolkit
python main.py
```
